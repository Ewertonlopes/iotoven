# IoT REFLOW OVEN

A low cost embedded device to do SMD soldering

## Hardware Required

- Esp32 devkit.
- High Potency Resistance.
- High temperature resistant material.
- Temperature Sensor.
- PWM actuator.

## Control Theory

Using a digital PID controller. 

## How to Use 

Just run the system as is in a esp32 and connect via STA.
